Projets


Projet en apprentissage machine

  • Développer un algorithme d'apprentissage machine pour la classification précise de deux variétés de riz : Cammeo et Osmancik.
  • Classer les grains de riz en se fondant sur l'analyse de sept caractéristiques morphologiques distinctes.
  • Évaluer différentes méthodologies telles que la régression logistique, les réseaux de neurones et les machines à vecteurs de support (SVM) afin d'optimiser les performances.
  • Utiliser Python et Jupyter Notebook pour rédiger un rapport détaillé sur l'avancement du projet.



BlueJayDB

  • Développer les capacités de BlueJayDB, un logiciel qui intègre plusieurs bases de données en une seule, tout en gérant les différences entre les systèmes de gestion de bases de données (SGBD).
  • Écrire un code en Java qui implémente une validation robuste des requêtes SQL.
  • Décomposer les requêtes SQL en clauses et les adapter aux langages spécifiques de chaque SGBD.
  • Concevoir des tests unitaires pour vérifier que BlueJayDB effectue une validation syntaxique précise des requêtes SQL correctement formulées.

Réalisations

Modèle d'apprentissage machine

Conception d'un modèle d'apprentissage machine pour la prédiction de deux variétés de riz, fondé sur leurs caractéristiques morphologiques distinctes. Cette approche innovante facilite grandement la classification précise et efficiente entre les deux catégories Cammeo et Osmancik.

Technologies utilisées : Python, Numpy, Pandas, Scikit-learn, TensorFlow, Keras

Amélioration de BlueJayDB

Amélioration du logiciel BlueJayDB par l'ajout de fonctionnalités pour la validation des requêtes SQL et la décomposition de ces dernières.

Technologies utilisées : Java, NetBeans, JSQLParser, JUnit 4.
